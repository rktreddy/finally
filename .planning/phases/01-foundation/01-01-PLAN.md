    **Step 2: Create `backend/app/db/schema.sql`** (per D-05, D-08)
    Raw SQL file with all 6 tables from PLAN.md Section 7. Every statement uses `CREATE TABLE IF NOT EXISTS`. Tables:

    ```sql
    CREATE TABLE IF NOT EXISTS users_profile (
        id TEXT PRIMARY KEY DEFAULT 'default',
        cash_balance REAL NOT NULL DEFAULT 10000.0,
        created_at TEXT NOT NULL DEFAULT (datetime('now'))
    );

    CREATE TABLE IF NOT EXISTS watchlist (
        id TEXT PRIMARY KEY,
        user_id TEXT NOT NULL DEFAULT 'default',
        ticker TEXT NOT NULL,
        added_at TEXT NOT NULL DEFAULT (datetime('now')),
        UNIQUE(user_id, ticker)
    );

    CREATE TABLE IF NOT EXISTS positions (
        id TEXT PRIMARY KEY,
        user_id TEXT NOT NULL DEFAULT 'default',
        ticker TEXT NOT NULL,
        quantity REAL NOT NULL,
        avg_cost REAL NOT NULL,
        updated_at TEXT NOT NULL DEFAULT (datetime('now')),
        UNIQUE(user_id, ticker)
    );

    CREATE TABLE IF NOT EXISTS trades (
        id TEXT PRIMARY KEY,
        user_id TEXT NOT NULL DEFAULT 'default',
        ticker TEXT NOT NULL,
        side TEXT NOT NULL CHECK(side IN ('buy', 'sell')),
        quantity REAL NOT NULL,
        price REAL NOT NULL,
        executed_at TEXT NOT NULL DEFAULT (datetime('now'))
    );

    CREATE TABLE IF NOT EXISTS portfolio_snapshots (
        id TEXT PRIMARY KEY,
        user_id TEXT NOT NULL DEFAULT 'default',
        total_value REAL NOT NULL,
        recorded_at TEXT NOT NULL DEFAULT (datetime('now'))
    );

    CREATE TABLE IF NOT EXISTS chat_messages (
        id TEXT PRIMARY KEY,
        user_id TEXT NOT NULL DEFAULT 'default',
        role TEXT NOT NULL CHECK(role IN ('user', 'assistant')),
        content TEXT NOT NULL,
        actions TEXT,
        created_at TEXT NOT NULL DEFAULT (datetime('now'))
    );
    ```

    **Step 3: Create `backend/app/db/seed.py`** (per D-07)
    Function `seed_defaults(db: aiosqlite.Connection) -> None` that:
    - Uses `INSERT OR IGNORE INTO users_profile (id, cash_balance) VALUES ('default', 10000.0)`
    - Loops over `DEFAULT_TICKERS = ["AAPL", "GOOGL", "MSFT", "AMZN", "TSLA", "NVDA", "META", "JPM", "V", "NFLX"]`
    - For each ticker: `INSERT OR IGNORE INTO watchlist (id, user_id, ticker) VALUES (uuid4(), 'default', ticker)`
    - Calls `await db.commit()` at the end
    - Uses `from __future__ import annotations` and `import uuid`
    - Module docstring, `logging.getLogger(__name__)`

    **Step 4: Create `backend/app/db/__init__.py`** (per D-01, D-02, D-03, D-06)
    Exports: `init_db`, `get_db_path`
    - `get_db_path() -> str`: Returns `os.environ.get("DB_PATH", "db/finally.db")`
    - `init_db(db_path: str) -> aiosqlite.Connection`:
      1. Ensure parent directory exists: `Path(db_path).parent.mkdir(parents=True, exist_ok=True)`
      2. `db = await aiosqlite.connect(db_path)`
      3. `db.row_factory = aiosqlite.Row` (per Pitfall 6)
      4. `await db.execute("PRAGMA journal_mode=WAL")` (per D-02)
      5. `await db.execute("PRAGMA foreign_keys=ON")`
      6. Read schema.sql: `schema_path = Path(__file__).parent / "schema.sql"`; `schema_sql = schema_path.read_text()`
      7. `await db.executescript(schema_sql)` — DDL only, no data (per Pitfall 1)
      8. `from .seed import seed_defaults`; `await seed_defaults(db)`
      9. Return `db`
    - Follow existing patterns: `from __future__ import annotations`, docstring, `__all__ = ["init_db", "get_db_path"]`

    IMPORTANT anti-pattern per research: Do NOT use `executescript()` for seed data inserts. Use `executescript()` only for DDL (schema.sql), then use individual `execute()` calls in seed.py for data inserts.
