    - `async def get_watchlist_tickers(db: aiosqlite.Connection, user_id: str = "default") -> list[dict]` — SELECT ticker, added_at FROM watchlist WHERE user_id = ? ORDER BY added_at; return list of dicts via `dict(row)`.

    Portfolio/trade functions (for Plan 02 — define them now to avoid file conflicts):
    - `async def get_user_cash(db, user_id="default") -> float` — SELECT cash_balance FROM users_profile WHERE id = ?; return float.

    - `async def get_position(db, ticker, user_id="default") -> dict | None` — SELECT from positions WHERE user_id = ? AND ticker = ?; return dict or None.

    - `async def delete_position(db, ticker, user_id="default") -> None` — DELETE FROM positions WHERE user_id = ? AND ticker = ?.
    - `async def insert_trade(db, ticker, side, quantity, price, user_id="default") -> dict` — INSERT INTO trades (...) VALUES (...). Return dict with id, ticker, side, quantity, price, executed_at.
    - `async def insert_snapshot(db, total_value, user_id="default") -> None` — INSERT INTO portfolio_snapshots (id, user_id, total_value) VALUES (?, ?, ?). Commit.
    - `async def get_snapshots(db, user_id="default", limit=500) -> list[dict]` — SELECT total_value, recorded_at FROM portfolio_snapshots WHERE user_id = ? ORDER BY recorded_at ASC LIMIT ?; return list of dicts.

    IMPORTANT: Do NOT call `await db.commit()` in functions used within a transaction (update_cash_balance, upsert_position, delete_position, insert_trade).     - `from __future__ import annotations`
    - Module-level logger
    - `router = APIRouter(tags=["watchlist"])`
    - Pydantic model: `class AddTickerRequest(BaseModel): ticker: str`

    `@router.get("/watchlist")`:
    - Get db and cache from `request.app.state`
    - Call `get_watchlist_tickers(db)`
    - For each ticker, call `cache.get(ticker)` to get PriceUpdate
    - Return list of dicts: `{"ticker": ..., "added_at": ..., "price": ..., "change": ..., "change_percent": ..., "direction": ...}` (per D-13). If no price in cache, set price fields to None.

    `@router.post("/watchlist", status_code=201)`:
    - Parse `AddTickerRequest` body
    - Normalize: `ticker = body.ticker.strip().upper()` (per D-10)
    - Get db, source from `request.app.state`
    - Try `add_watchlist_ticker(db, ticker)`. Catch `sqlite3.IntegrityError` and raise `HTTPException(status_code=409, detail="Ticker already in watchlist")` (per D-12)
    - Call `await source.add_ticker(ticker)` (per D-10, WL-04)
    - Return the created ticker dict

    `@router.delete("/watchlist/{ticker}")`:
    - Normalize: `ticker = ticker.upper()` (per D-10)
    - Get db, source, cache from `request.app.state`
    - Call `remove_watchlist_ticker(db, ticker)`. If returns 0, raise `HTTPException(status_code=404, detail="Ticker not in watchlist")` (per D-12)
    - Call `await source.remove_ticker(ticker)` (per D-11)
    - Call `cache.remove(ticker)` (per D-11)
    - Return `{"status": "removed", "ticker": ticker}`
