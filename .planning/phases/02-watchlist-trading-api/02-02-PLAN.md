    id TEXT PRIMARY KEY,
    user_id TEXT NOT NULL DEFAULT 'default',
    ticker TEXT NOT NULL,
    quantity REAL NOT NULL,
    avg_cost REAL NOT NULL,
    updated_at TEXT NOT NULL DEFAULT (datetime('now')),

);

    **1. Create `backend/app/routes/portfolio.py`**:

    ```
    from __future__ import annotations
    import asyncio, logging, uuid
    from fastapi import APIRouter, HTTPException, Request
    from pydantic import BaseModel
    from app.db.repository import (get_user_cash, get_positions, get_position,
        update_cash_balance, upsert_position, delete_position,
        insert_trade, insert_snapshot, get_snapshots)
    ```

    Module-level logger. `router = APIRouter(tags=["portfolio"])`

    **Pydantic models:**
    - `class TradeRequest(BaseModel): ticker: str; quantity: float; side: str` — side must be "buy" or "sell"

    **`@router.get("/portfolio")`** (per D-07, D-08, D-19, PT-01):
    - Get db, cache from request.app.state
    - `cash = await get_user_cash(db)`
    - `positions = await get_positions(db)`
    - For each position, compute:
      - `current_price = cache.get_price(pos["ticker"])` — use 0.0 if None
      - `unrealized_pnl = (current_price - pos["avg_cost"]) * pos["quantity"]` (per D-07)
      - `pnl_percent = ((current_price - pos["avg_cost"]) / pos["avg_cost"] * 100) if pos["avg_cost"] > 0 else 0.0` (guard division by zero)
    - `total_value = cash + sum(pos["quantity"] * (cache.get_price(pos["ticker"]) or 0.0) for pos in positions)` (per D-08)
    - `total_pnl = sum of all unrealized_pnl`
    - Return `{"cash": cash, "positions": enriched_list, "total_value": total_value, "total_pnl": total_pnl}`

    **`@router.post("/portfolio/trade")`** (per D-01 through D-06, D-09, D-15, D-18, PT-02 through PT-06, PT-08):
    - Parse TradeRequest body
    - Normalize: `ticker = body.ticker.strip().upper()`
    - Validate side: if `body.side not in ("buy", "sell")`, raise HTTPException(400, "Side must be 'buy' or 'sell'")
    - Validate quantity: if `body.quantity <= 0`, raise HTTPException(400, "Quantity must be positive")
    - Get db, cache from request.app.state
    - `current_price = cache.get_price(ticker)` — if None, raise HTTPException(400, f"No price available for {ticker}") (per D-01)
    - `cash = await get_user_cash(db)`
    - `total_cost = body.quantity * current_price`

    **If side == "buy":**
    - If `total_cost > cash`, raise HTTPException(400, "Insufficient cash") (per D-03, PT-04)
    - `new_cash = cash - total_cost`
    - Get existing position: `pos = await get_position(db, ticker)`
    - If pos exists: `new_qty = pos["quantity"] + body.quantity`; `new_avg = (pos["quantity"] * pos["avg_cost"] + body.quantity * current_price) / new_qty` (per D-09)
    - If no pos: `new_qty = body.quantity`; `new_avg = current_price`
    - `await update_cash_balance(db, new_cash)`
    - `await upsert_position(db, ticker, new_qty, new_avg)`
    - `trade = await insert_trade(db, ticker, "buy", body.quantity, current_price)`
    - `await db.commit()` (atomic transaction per D-02)

    **If side == "sell":**
    - `pos = await get_position(db, ticker)` — if None, raise HTTPException(400, "No position in {ticker}") (per PT-05)
    - If `body.quantity > pos["quantity"] + 1e-9`, raise HTTPException(400, "Insufficient shares") (per D-03, PT-05, epsilon tolerance per pitfall 6)
    - `sell_qty = min(body.quantity, pos["quantity"])` (clamp for floating point)
    - `new_cash = cash + sell_qty * current_price`
    - `remaining = pos["quantity"] - sell_qty`
    - `await update_cash_balance(db, new_cash)`
    - If `remaining < 1e-9`: `await delete_position(db, ticker)` (per D-04)
    - Else: `await upsert_position(db, ticker, remaining, pos["avg_cost"])` (per D-09 — sells do not change avg_cost)
    - `trade = await insert_trade(db, ticker, "sell", sell_qty, current_price)`
    - `await db.commit()` (atomic per D-02)

    **After successful commit (both buy/sell):**
    - Record immediate snapshot (per D-15, PT-08): compute total_value from new cash + all positions * live prices, call `await insert_snapshot(db, total_value)`
    - Return `{"trade": trade, "cash_balance": new_cash}` (per D-18)

    **`@router.get("/portfolio/history")`** (per PT-09):
    - `snapshots = await get_snapshots(db)`
    - Return `{"snapshots": snapshots}`

    **`async def snapshot_loop(app)`** (per D-14, D-16, PT-07):
    - `logger = logging.getLogger(__name__)`
    - `while True:` loop
    - `await asyncio.sleep(30)`
    - `try:` compute total_value from `get_user_cash(app.state.db)` + positions * prices from `app.state.cache.get_price()`
    - Call `await insert_snapshot(app.state.db, total_value)`
    - `except asyncio.CancelledError: logger.info("Snapshot task cancelled"); raise`
    - `except Exception: logger.exception("Error recording portfolio snapshot")`

    **`async def record_snapshot(db, cache, user_id="default")`** — shared helper used by both the background task and post-trade inline:
    - Get cash from DB, get all positions, compute total_value = cash + sum(qty * price for each position)
    - Call `await insert_snapshot(db, total_value, user_id)`
