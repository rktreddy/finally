4. Create `frontend/lib/types.ts` with TypeScript interfaces matching the backend API shapes exactly:
   - `PriceUpdate`: ticker, price, previous_price, timestamp (number), change, change_percent, direction ("up"|"down"|"flat")
   - `PriceMap`: Record<string, PriceUpdate>
   - `WatchlistItem`: ticker, added_at (string), price (number|null), change (number|null), change_percent (number|null), direction (string|null)
   - `Position`: ticker, quantity, avg_cost, current_price, unrealized_pnl, pnl_percent
   - `Portfolio`: cash, positions (Position[]), total_value, total_pnl
   - `TradeResult`: trade ({id, ticker, side, quantity, price, executed_at}), cash_balance
   - `PortfolioSnapshot`: total_value, recorded_at (string)
   - `ChatResponse`: message (string), actions ({trades: Array<{ticker, side, quantity, price, status}>, watchlist_changes: Array<{ticker, action, status}>, errors: string[]})
   - `ConnectionStatus`: "connected" | "reconnecting" | "disconnected"

5. Create `frontend/lib/api.ts` with fetch wrappers (per D-04, D-27, D-28):
   - `fetchWatchlist(): Promise<WatchlistItem[]>` -- GET /api/watchlist
   - `addTicker(ticker: string)` -- POST /api/watchlist with body {ticker}
   - `removeTicker(ticker: string)` -- DELETE /api/watchlist/{ticker}
   - `fetchPortfolio(): Promise<Portfolio>` -- GET /api/portfolio
   - `executeTrade(ticker: string, quantity: number, side: "buy"|"sell"): Promise<TradeResult>` -- POST /api/portfolio/trade
   - `fetchPortfolioHistory(): Promise<{snapshots: PortfolioSnapshot[]}>` -- GET /api/portfolio/history
   - `sendChatMessage(message: string): Promise<ChatResponse>` -- POST /api/chat with body {message}
   All functions use same-origin fetch (empty base URL). Throw errors with status info on non-ok responses. For trade errors, parse response body for `detail` field.

6. Create `frontend/hooks/useSSE.ts` (per D-24, D-25, D-26, D-29):
   - `"use client"` directive at top
   - Custom hook `useSSE()` returning `{ prices: PriceMap, status: ConnectionStatus, getHistory: (ticker: string) => {time: number, value: number}[] }`
   - Creates `new EventSource("/api/stream/prices")` in useEffect
   - `onopen` sets status to "connected"
   - `onmessage` parses JSON, updates `prices` state (setPrices), accumulates per-ticker history in a `useRef<Record<string, {time: number, value: number}[]>>({})`
   - `onerror` sets status to "reconnecting"
   - Cleanup calls `es.close()` and sets status to "disconnected"
   - Cap history arrays at 200 data points per ticker (shift old entries when limit reached)
   - `getHistory` is wrapped in `useCallback` returning the ref array for a given ticker
