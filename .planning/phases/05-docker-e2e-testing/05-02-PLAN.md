Key pages and API endpoints to test against:
- GET http://localhost:8000 — serves the frontend SPA
- GET /api/stream/prices — SSE stream of price updates
- GET /api/watchlist — returns watchlist with prices
- POST /api/watchlist — add ticker {ticker: string}
- DELETE /api/watchlist/{ticker} — remove ticker
- GET /api/portfolio — returns positions, cash, total value
- POST /api/portfolio/trade — execute trade {ticker, quantity, side}
- POST /api/chat — send message {message: string}, returns {message, trades, watchlist_changes}

LLM mock mode (LLM_MOCK=true) returns deterministic responses — the chat test should send a message and verify a response appears.

Default seed data:
- 10 watchlist tickers: AAPL, GOOGL, MSFT, AMZN, TSLA, NVDA, META, JPM, V, NFLX
- Cash balance: $10,000.00
- No initial positions
