   - `async def execute_actions(chat_response: ChatResponse, db, cache, source) -> dict:` that returns `{"trades": [], "watchlist_changes": [], "errors": []}`
   - For each trade in `chat_response.trades`:
     - `ticker = trade.ticker.strip().upper()`
     - Validate side is "buy" or "sell"
     - Get `current_price = cache.get_price(ticker)` — if None, append error "No price available for {ticker}" and continue
     - If buy: `cash = await get_user_cash(db)`, `total_cost = trade.quantity * current_price`, if `total_cost > cash` append error "Insufficient cash for {ticker}" and add trade result with status="failed", continue. Otherwise: compute new_avg (get existing position, calculate weighted average), `await update_cash_balance(db, cash - total_cost)`, `await upsert_position(db, ticker, new_qty, new_avg)`, `await insert_trade(db, ticker, "buy", trade.quantity, current_price)`, `await db.commit()`, `await record_snapshot(db, cache)`, append trade result with status="executed" and price=current_price
     - If sell: get position, if None or quantity insufficient append error and trade with status="failed", continue. Otherwise: compute sell proceeds, `await update_cash_balance(db, cash + sell_qty * current_price)`, handle position delete vs update (same epsilon pattern as portfolio.py: `remaining < 1e-9` -> delete), `await insert_trade(db, ticker, "sell", sell_qty, current_price)`, `await db.commit()`, `await record_snapshot(db, cache)`, append trade result with status="executed"
   - For each wl_change in `chat_response.watchlist_changes`:
     - `ticker = wl_change.ticker.strip().upper()`
     - If action == "add": try `await add_watchlist_ticker(db, ticker)` then `await source.add_ticker(ticker)`, append result with status="executed". Catch `sqlite3.IntegrityError` -> append with status="failed", error="Already in watchlist"
     - If action == "remove": `deleted = await remove_watchlist_ticker(db, ticker)`, if deleted == 0 append status="failed" error="Not in watchlist", else `await source.remove_ticker(ticker)`, `cache.remove(ticker)`, append status="executed"
   - Return the results dict

   Portfolio context assembly helper:
   - `async def assemble_portfolio_context(db, cache) -> dict:` that returns `{"cash": float, "positions": list[dict], "watchlist": list[dict], "total_value": float}`
   - Get cash via `get_user_cash(db)`
   - Get positions via `get_positions(db)`, enrich each with `current_price = cache.get_price(pos["ticker"]) or 0.0` and `unrealized_pnl = (current_price - pos["avg_cost"]) * pos["quantity"]`
   - Get watchlist via `get_watchlist_tickers(db)`, enrich each with `price = cache.get_price(item["ticker"])`
   - Compute total_value = cash + sum of (quantity * current_price) for all positions
